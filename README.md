# SIDCU - Simulador de Carreras Universitarias

> Plataforma de simulación interactiva para explorar y diseñar tu carrera académica universitaria

---

## 🎯 Descripción

SIDCU es un simulador integral que te permite explorar diferentes carreras universitarias, visualizar rutas académicas personalizadas y tomar decisiones informadas sobre tu futuro educativo. Experimenta con diferentes combinaciones de cursos, especialidades y horarios antes de comprometerte.

---

## ✨ Características

- 🎓 **Catálogo completo** de carreras y programas universitarios
- 📅 **Simulación de horarios** y cargas académicas
- 📊 **Análisis de rutas académicas** personalizadas
- 💼 **Proyección laboral** basada en carrera elegida
- 🎯 **Recomendaciones** basadas en intereses y aptitudes
